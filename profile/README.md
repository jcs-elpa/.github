**jcs-elpa** is ELPA for [jcs-emacs](https://github.com/jcs-emacs/jcs-emacs), using github-elpa

### Quickstart

```sh
(setq package-archives
      `(,@package-archives
        ("jcs-elpa" . "https://jcs-emacs.github.io/jcs-elpa/packages/")))
```
